# Finnis-Sinclair EAM Example

This folder contains a binary Finnis-Sinclair EAM potential:

- `initial.toml`
- `forgeff.train.toml`

The `initial.toml` file uses species-pair density tables for Al-Cu. The
training setting points to `../../data/binary/training.cfg`, so you can replace
that path with your own configuration list before running a real fit.
