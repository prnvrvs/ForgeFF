# Double Morse Pairwise Example

This folder contains a built-in double-Morse pair potential:

- `initial.toml`
- `forgeff.train.toml`

The `initial.toml` file shows a slightly richer analytical pair form with two
Morse wells and a constant shift. The training setting points to
`../../data/unary/training.cfg`, so you can replace that path with your own
configuration list before running a real fit.
