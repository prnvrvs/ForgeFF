ForgeFF Examples
================

This gallery groups the source examples by style:

- ``examples/toml`` contains the copyable TOML training examples and the
  direct Python training launcher.
- ``examples/python`` is the Python-based tutorial side. Its example folders
  mirror the TOML layout under ``examples/python/pairwise/``,
  ``examples/python/eam/``, and ``examples/python/adp/``.

Use the TOML folder for copyable training settings and the Python gallery for
the narrated walkthroughs.
